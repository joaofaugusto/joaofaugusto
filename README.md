Trabalho com arquitetura de sistemas, automação de processos e integração entre serviços. Atualmente lidero o desenvolvimento do **InsuranceHub** pela @Vime-Sistemas e do **Flow** pela @Setup-Assessoria, plataformas voltadas para assessorias e corretores de seguros.

## Stack principal

* **Backend:** Go, Python e Node.js
* **Frontend:** React, JavaScript, TypeScript, Vite
* **Banco de dados:** PostgreSQL
* **Infra:** Docker e Nginx

## Interesses técnicos
System design, microserviços, segurança de API e escalabilidade em ambientes distribuídos.

## Projetos em destaque
* **InsuranceHub** – Plataforma de gestão e análise de seguros.
* **Flow** – Plataforma de gestão de negócios para corretores de seguros.
